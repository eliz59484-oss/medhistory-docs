## 📱 Мобильная разработка с Expo

### Что такое Expo
- Фреймворк для создания мобильных приложений (iOS + Android)
- Работает поверх React Native
- Позволяет тестировать на симуляторе или реальном телефоне через QR-код

### Установка и запуск
```
Запрос Antigravity: "Установи субагентов для разработки приложений 
в Expo. Разрабатывать будем на Windows в симуляторе Android."
```

**Необходимо:**
1. Установить Android Studio (~21 ГБ)
2. Настроить Android SDK
3. Запустить эмулятор Android
4. Установить Expo CLI

### Архитектура приложения
```
Фронтенд (экраны) → Бэкенд (API, данные) → Деплой (TestFlight / Google Play)
```

**Правильный порядок разработки:**
1. Идея → MindMap
2. Wireframing (прорисовка экранов)
3. Фронтенд (визуал)
4. Бэкенд (логика, интеграции)
5. Тестирование (QR-код → телефон)
6. Деплой + модерация

### Прорисовка экранов
> Используйте холст (Google Slides, Figma, Stitch) — нарисуйте каждый экран:
> - Экран 1: Регистрация (логин, пароль)
> - Экран 2: Главная страница (элементы, логотип)
> - Экран 3–N: Основной функционал

### Пример: Приложение-трекер маршрутов
Идея от участника:
- Запоминает маршрут на **3D-карте**
- Фиксирует **фотографии** по маршруту
- Работает **без интернета**

### Деплой в магазины
- **Модерация Apple/Google** — люди проверяют код
- Могут давать ТЗ на исправление → повторная отправка
- Через **TestFlight** можно раздать приложение тестировщикам (иконка на телефоне)

## 🎨 Stitch — инструмент для дизайна (MCP)

### Что это
- Онлайн-сервис для генерации дизайна интерфейсов
- Подключается к Antigravity через **MCP-сервер**
- Использует нейросеть (бесплатная, качество на уровне платных)

### Как работает с Antigravity
```
Шаг 1: В Antigravity пишете промпт → "Используя MCP Stitch, 
        сделай мне 3 варианта дизайна (3x)"
Шаг 2: Заходите в Stitch → видите готовый проект с дизайном
Шаг 3: Выбираете лучший вариант
Шаг 4: Экспорт из Stitch → импорт в Antigravity
```

### Возможности
- 3 варианта дизайна одним запросом (функция **3x**)
- Автогенерация **brand guide** (цвета, шрифты, стили)
- Редактирование каждого окна отдельно
- Работает для сайтов и мобильных приложений

## 🔗 Claude Code ↔ Telegram (мост)

### Зачем
- Управление проектами **с телефона**, без открытого IDE
- Дублирование терминала Claude Code в Telegram-бот

### Выбор репозитория
Георгий провёл сравнение мостов на GitHub:

| Репозиторий | Звёзды | Последний коммит | Платформа |
|-------------|--------|------------------|-----------|
| **CC Connect** | 104 | 13.03.2026 | Windows ✅ |
| CloudGram | 2100 | 03.04 (давно) | Mac ✅ |

> **Для Windows** выбран **CC Connect** (свежий, активно обновляется)
> **Для Mac** — **CloudGram** (больше звёзд, больше функционала)

### Установка (на примере Сергея)
1. Запрос в Antigravity: *"Найди качественную директорию в GitHub для мостика между Claude Code и Telegram"*
2. Выбрать репозиторий → зелёная кнопка "Use" → развернуть
3. Получить **Telegram Bot Token** (новый!)
4. Настроить → запустить

### Результат
- Бот работает **даже при закрытом IDE** (если настроен автозапуск)
- Можно давать запросы из Telegram → Claude Code выполняет
- Команды: `/teleport`, `/start`, `/stop`, `/cancel`

## 💰 Продуктовая стратегия и монетизация

- **Решение:** Автоматизация через бота
- **Оценка:**
  - Агентство: 15–30K € (1.5–3 млн ₽)
  - Фрилансер: 7–15K €
  - Самостоятельно (вайбкодинг): значительно дешевле

### Как продавать решения
1. **Переговоры** — понять все процессы клиента
2. **Коммерческое предложение** — красивое, с оценкой по срокам и стоимости
3. **Оплата в 2–3 этапа:**
   - Аванс 35–40%
   - MVP (работающий продукт)
   - Финальная сдача
4. **Бонус:** Оценить все бизнес-процессы + дорожная карта на год
5. **Оффер:** Подписка на внедрение по дорожной карте

## 🏠 Разное (важные заметки)

- **Кириллица в путях Windows** — вызывает конфликты при установке (решение: переименовать папки)
- **Скрытая папка Claude Code** — начинается с точки (`.claude`), на Windows не видна в обычных файловых менеджерах
- **Глобальное правило** — можно прописать: *"Если нужна команда в терминале — скидывай мне скрипт, я ввожу"* (для тех, у кого проблемы с терминалом)
- **Переключение между задачами** — компании теряют до 3 месяцев в год на ментальное переключение (методология критической цепи)
- **Практика каждый день** — ключ к закреплению навыков вайбкодинга

### По приложениям
- Мнение: приложения через 2 года изменятся (очки, новые интерфейсы)
- Но навык разработки приложений → полезен для заказчиков уже сейчас
